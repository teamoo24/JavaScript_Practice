構文

  • Math.round(四捨五入する数字)
  • Math.floor(切り捨てる数字)
  • Math.ceil(切り上げる数字)
  • Math.trunc(整数部分だけを求めてる数字)

説明

四捨五入や切り捨てにはMath.round()、Math.floor()、Math.ceil()、Math.trunc()メソッドを利用します。

サンプル

round_6_24 = () => Math.round(6.24)
floor_6_24 = () => Math.floor(6.24)
ceil_6_24 = () => Math.ceil(6.24)
trunc_6_24 = () => Math.trunc(6.24)
round_7_8 = () => Math.round(7.8)
floor_7_8 = () => Math.floor(7.8)
ceil_7_8 = () => Math.ceil(7.8)
trunc_7_8 = () => Math.trunc(7.8)
