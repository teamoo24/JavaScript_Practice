(引数)=>{処理内容}

関数を定義する方法は、function宣言の他にアロー関数を使う

使い方

const calcSum = (a,b,c) => {

const result = a+b+c;

return result;

};

実行結果

アロー関数は、functionと異なりいくつかの省略記法がある。

引数が一つの場合は省略できる。

使い方

const myFunction1 = (a) => {
return a+2;
}

const myFunction2 = a => {
return a+2;
}

この場合myFunction1、myFunction2は同じ作動をする

実行結果

アロー 関数 内 の 処理 が 1 行 の とき は、{} と return を 省略 でき ます。

使い方

const myFunction3 = a => a+2

実行結果
